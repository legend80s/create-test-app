<h1 align="center">Welcome to create-test-app 👋</h1>

> A scaffolding CLI to make your project tested by Jest and with coverage enabled.

## Use

For TypeScript project.

```sh
npx create-test-app --type ts
```

For JavaScript project.

```sh
npx create-test-app
```

## Run tests

```sh
npm test
```
